TeamTaskManager helps team to manage and assign task to team members .Team Task Manager Web application developed using .NET Core Web API and SQL Server for the backend, and Angular, TypeScript, Bootstrap, and HTML/CSS for the frontend. The application allows users to manage tasks effectively with features like adding, viewing, and deleting users, creating and tracking tasks across ToDo, InProgress, and Done stages.

Frontend of this project is written in Angular github repository link : https://github.com/CHAVDAYASHVANTSINH612/TeamTaskManager_FrontEnd_Angular
